# Birthday surprise page

Static site for GitHub Pages (`index.html` at repo root).

## Live site (after you enable Pages)

**URL:** https://zkyko.github.io/birthday/

## Enable GitHub Pages once

1. Open [github.com/zkyko/birthday](https://github.com/zkyko/birthday).
2. Go to **Settings** → **Pages** (left sidebar).
3. Under **Build and deployment** → **Source**, choose **Deploy from a branch**.
4. Set **Branch** to **`main`** and folder **`/ (root)`**, then click **Save**.
5. Wait ~1 minute; refresh the Pages section — it shows the site URL when ready.
